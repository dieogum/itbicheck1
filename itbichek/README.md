# ITBI Check

Landing page para diagnóstico gratuito de ITBI em São Paulo.

## Como usar

Abra o arquivo `index.html` no navegador para visualizar localmente.

## Deploy

Hospedado via Vercel. Qualquer push na branch `main` atualiza o site automaticamente.

## Antes de publicar

Substitua `5511900000000` no `index.html` pelo número real de WhatsApp Business.
